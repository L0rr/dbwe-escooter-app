# DBWE E-Scooter App

Praxisarbeit im Modul DBWE.  
Die Applikation basiert auf Flask und PostgreSQL und bildet eine einfache E-Scooter-Verleihplattform ab.

## Voraussetzungen
- Python 3
- PostgreSQL
- pip

## Installation
1. Projekt entpacken oder klonen
2. Virtuelle Umgebung erstellen
3. Abhängigkeiten installieren
4. Datenbank konfigurieren
5. Seed-Datei ausführen
6. Anwendung starten

## Hinweis
Die im Repository enthaltene Seed-Datei wurde absichtlich mit neutralen Beispiel-Zugangsdaten angepasst, damit keine identischen Login-Daten der bereitgestellten Anwendung veröffentlicht werden.
